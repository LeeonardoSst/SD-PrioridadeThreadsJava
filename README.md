# SD-PrioridadeThreadsJava
Atividade Sábado Letivo

Repositório com código para atividade de Sistemas Distribuídos / Threads em Java.
Contém o programa `LancadorPrioridade.java` que demonstra execução/controlador de prioridades
entre threads. Instruções para compilação e execução estão neste README.

Relatório da atividade: 
Foi incluido o arquivo "LancadorPrioridade.java". Em seguida, no code workspace, o mesmo foi compilado e depois executado.
Comando de compilação: javac LancadorPrioridade.java
Comando de execução: java LancadorPrioridade > LancadorPrioridade_$(date +%Y%m%d_%H%M%S).log &
    Resultando em 4 arquivos, AltaPrioridade.class, BaixaPrioridade.class, LancadorPrioridade.class e LancadorPrioridade.log

    No arquivo log é possivel analisar quais são executados por primeiro.
